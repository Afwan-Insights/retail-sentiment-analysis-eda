# E-Commerce Clothing Reviews: Exploratory Data Analysis (EDA)

## Project Overview
The goal of this project is to analyze customer reviews from a women's clothing e-commerce store to identify the root causes of negative feedback and uncover business opportunities. 

## Tech Stack Used
* **Python:** Data manipulation and analysis.
* **Pandas:** Data cleaning, handling missing values, and filtering.
* **Matplotlib & Seaborn:** Statistical data visualization.

## Key Business Insights
1. **Sizing Discrepancies:** The 'Trend' department suffers the lowest average rating. Text analysis of 1-star reviews reveals customers are consistently reporting garments are drastically oversized compared to the label. *Recommendation: Audit sizing charts and update the website to advise sizing down.*
2. **Age is Not a Factor:** Bivariate analysis shows the median customer age (40-42) remains identical across all 1-star to 5-star reviews. Dissatisfaction is tied to the physical product, not demographic preferences.
3. **Quality vs. Price Mismatch:** Negative reviews frequently cite materials feeling "cheap" or "itchy," indicating that the current pricing strategy for the Trend line does not align with the material quality.
